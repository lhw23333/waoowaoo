# Waoowaoo AI 影视 Studio — 内容创作操作手册

> 从一段小说文字到一部完整 AI 短剧视频的全流程指南

---

## 一、创作流程总览

```
小说/故事文本
    │
    ▼
┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐
│ 1.导入故事 │──▶│ 2.剧本生成 │──▶│ 3.资产制作 │──▶│ 4.分镜制作 │
│  粘贴文本  │   │  AI 分析  │   │ 角色/场景 │   │ 图片生成  │
└──────────┘   └──────────┘   └──────────┘   └──────────┘
                                                    │
    ┌──────────┐   ┌──────────┐   ┌──────────┐     │
    │ 7.导出成片 │◀──│ 6.视频生成 │◀──│ 5.AI配音  │◀────┘
    │  打包下载  │   │ 动态视频  │   │ 多角色TTS │
    └──────────┘   └──────────┘   └──────────┘
```

## 二、前置准备

### 2.1 注册账号

1. 打开浏览器访问 `http://localhost:3000`
2. 页面自动跳转至登录页，点击 **注册**
3. 填写：手机号（作为用户名）、密码（至少 6 位）、确认密码
4. 注册成功后自动跳转至登录页，输入账号密码登录

### 2.2 配置 AI 服务

> 首次使用必须完成此步骤，否则无法调用 AI 功能。

1. 登录后点击右上角 **用户头像** → **个人设置**
2. 进入 **API 配置** 页签
3. 按以下优先级配置 API Key：

**必配（完成核心流程）**：

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 添加 LLM 提供商 | 填入 BaseURL 和 API Key（推荐 OpenRouter） |
| 2 | 添加 LLM 模型 | 选择类型"文本"，填入模型 ID |
| 3 | 添加图片提供商 | 推荐 fal.ai 或 Google AI |
| 4 | 添加图片模型 | 选择类型"图片"，绑定到上述提供商 |
| 5 | 添加视频提供商 | 推荐 fal.ai 或火山引擎 |
| 6 | 添加视频模型 | 选择类型"视频"，绑定到上述提供商 |

**选配（启用配音和口型同步）**：

| 步骤 | 操作 |
|------|------|
| 7 | 添加阿里百炼 API Key → 启用 AI 配音 |
| 8 | 添加 fal.ai 口型同步模型 → 启用唇形同步 |

4. 配置完成后，可使用 **测试连接** 按钮验证各 API Key 是否有效

## 三、完整创作流程

### 第一步：创建项目 & 导入故事

1. 在首页点击 **新建项目**，输入项目名称
2. 进入项目工作台，默认处于 **故事输入** 阶段
3. 在文本框中 **粘贴你的小说/剧本文本**
   - 支持长篇文本，系统会自动分集
   - 右下角显示字数统计
4. 设置创作参数：
   - **画面比例**：选择视频比例（9:16 竖屏 / 16:9 横屏 / 1:1 等）
   - **画风**：选择四种风格之一
     - 漫画风（美式漫画）
     - 精致国漫
     - 日系动漫风
     - 真人风格
   - **旁白开关**：是否包含旁白配音
5. 点击 **开始创作**

> **智能分集**：如果文本较长，系统会自动启动 AI 分集功能，将长篇内容拆分为多个集数。

### 第二步：剧本确认

AI 会自动完成以下分析：
- 将原文转换为专业剧本格式（动作描写、对白、旁白）
- 提取所有角色及其特征
- 识别所有场景/地点
- 按剧情节点分割为多个片段（Clip）

**你可以做的操作**：
- 浏览和编辑每个片段的剧本内容
- 在左侧切换不同集数
- 确认角色和场景列表无误
- 如果对分析结果不满意，可以修改文本后重新生成

### 第三步：角色 & 场景资产制作

进入 **资产管理** 界面，为每个角色和场景生成参考图片。

**角色资产**：
1. 系统已从剧本中提取角色列表
2. 对每个角色，点击 **生成形象** → AI 生成多张候选图
3. 从候选图中 **选择最满意的一张** 作为参考图
4. 可以编辑角色的外观描述词，然后重新生成
5. 如果需要修图，点击 **编辑** 进入图片修改界面

**场景资产**：
1. 同样，每个场景可以生成多张候选图
2. 选择最匹配的场景图作为参考
3. 可编辑场景描述后重新生成

**全局资产库**：
- 点击 **资产中心** 可管理跨项目复用的角色/场景/音色
- 在项目中可以从全局资产库"导入"已有角色/场景，实现多项目角色一致性
- 资产按文件夹分类管理

**批量操作**：
- **生成全部图片**：一键为所有未生成的角色/场景批量生成
- **重新生成全部**：对所有已有图片重新生成
- **全局分析**：让 AI 重新分析剧本提取角色/场景

> **Tips**：角色一致性是漫剧质量的关键。花时间在这一步选好参考图，后续分镜图片的角色一致性会更好。

### 第四步：分镜制作

进入 **分镜** 阶段，AI 已为每个片段生成分镜面板。

**分镜面板结构**：
每个分镜面板包含：
- 镜头编号和类型（特写、中景、远景等）
- 运镜方式（推、拉、摇、移等）
- 画面描述
- 角色和场景信息
- 图片提示词（Prompt）

**操作方式**：

| 操作 | 说明 |
|------|------|
| 生成全部图片 | 一键批量生成所有面板的分镜图 |
| 重新生成 | 对单个面板重新生成图片 |
| 生成变体 | 基于当前面板生成新的构图变体 |
| 编辑图片 | 进入修图模式调整细节 |
| 插入面板 | 在当前面板前/后插入新面板 |
| 删除面板 | 移除不需要的面板 |
| 拖拽排序 | 拖动面板调整顺序 |
| 查看 AI 数据 | 查看镜头类型、运镜、摄影规则、表演指导等详细信息 |
| 下载全部图片 | 打包下载所有分镜图片 |

**候选图选择**：
每次生成可能产生多张候选图，点击缩略图切换选择最佳的一张。

> **Tips**：分镜面板数量由剧本长度自动决定（大约每 15 个字对应 1 个面板）。如果觉得某段节奏需要调整，可以手动插入或删除面板。

### 第五步：AI 配音（可选）

> 需要提前配置阿里百炼（Qwen）API Key。

进入 **配音** 阶段：

1. **语音分析**：AI 自动分析剧本，将每句台词分配给对应角色
2. **音色设置**：
   - 为每个角色选择或设计专属音色
   - 可从全局音色库选择已有音色
   - 可使用 **AI 音色设计** 功能，描述想要的声音特点由 AI 生成
   - 可上传自定义音频作为参考音色
3. **生成配音**：
   - 逐句或批量生成配音
   - 可调整语速（默认 +50%）
   - 支持设置情感提示词和情感强度
4. **预览和调整**：
   - 点击播放按钮试听每句配音
   - 不满意的可以修改参数后重新生成

### 第六步：视频生成

进入 **视频** 阶段：

1. 系统显示所有分镜面板的视频生成状态
2. 点击 **生成全部视频** 或逐个面板生成
3. 生成模式：
   - **普通模式 (normal)**：从单张图片生成视频
   - **首末帧模式 (firstlastframe)**：指定首帧和末帧图片，生成过渡视频（画面更连贯）
4. 可选参数（取决于所选模型的能力）：
   - 分辨率：720p / 1080p
   - 时长：5s / 10s 等
   - 是否带音频（部分模型如 Seedance 1.5 Pro 支持）
5. 等待生成完成，预览每个片段的视频效果
6. 不满意可重新生成

> **Tips**：视频生成耗时较长（通常 30s - 3min 每个片段），建议批量生成后去做其他事情。可在 Bull Board（`localhost:3010/admin/queues`）查看队列进度。

### 第七步：口型同步（可选）

如果同时有配音和视频：
1. 系统提供口型同步功能
2. 将生成的语音与视频进行对齐
3. 角色说话时嘴型与语音同步

### 第八步：导出成片

1. 在视频阶段可以 **下载全部视频**（打包为 ZIP）
2. 也可以逐个面板下载单独的视频片段
3. 在分镜阶段可以 **下载全部图片**

> 项目支持 Remotion 视频编辑器，可以对最终视频进行剪辑和合成。

## 四、项目设置调优

在项目工作台右上角点击 **设置** 图标，可以调整：

### 4.1 画风设置

| 画风 | 描述 | 适合场景 |
|------|------|---------|
| 漫画风 | 美式漫画风格，线条分明 | 超级英雄、动作题材 |
| 精致国漫 | 精致的中国风漫画，色彩华丽 | 古风、仙侠、都市 |
| 日系动漫风 | 赛璐珞着色，日式动漫风 | 校园、恋爱、冒险 |
| 真人风格 | 电影级写实画面 | 都市剧、悬疑、现实题材 |

### 4.2 画面比例

| 比例 | 用途 |
|------|------|
| 9:16 | 竖屏短剧（抖音/快手/TikTok） |
| 16:9 | 横屏视频（B站/YouTube） |
| 1:1 | 方形视频（Instagram/小红书） |
| 其他 | 3:2、4:3、21:9 等特殊用途 |

### 4.3 模型选择

为项目的不同环节选择最合适的模型：
- **分析模型**：影响剧本分析和分镜规划质量，推荐使用较强的 LLM
- **角色/场景/分镜模型**：影响图片质量和一致性
- **视频模型**：影响视频质量和动态效果

## 五、效率技巧

### 5.1 批量操作

- 在每个阶段都有 **"生成全部"** 按钮，善用批量操作节省时间
- 批量任务通过异步队列执行，不会阻塞页面操作

### 5.2 资产复用

- 将常用角色/场景保存到 **全局资产库**
- 新项目可直接从资产库导入，保持角色一致性
- 音色也可以跨项目复用

### 5.3 任务监控

- 访问 `http://localhost:3010/admin/queues` 查看所有异步任务状态
- 四个队列：Image（图片）、Video（视频）、Voice（语音）、Text（文本）
- 可以查看失败任务的错误原因，手动重试

### 5.4 多集管理

- 长篇内容自动分集后，在工作台左侧切换集数
- 每集独立管理：剧本、分镜、配音、视频
- 角色和场景资产在全项目共享

### 5.5 迭代优化

- 每个环节都支持 **重新生成** 和 **手动编辑**
- 建议先快速跑完全流程预览效果，再逐步优化不满意的环节
- 图片和视频的 Prompt 都可以手动编辑后重新生成

## 七、常见创作问题

### Q1: 角色一致性不好怎么办？
- 在资产阶段花更多时间选择高质量的角色参考图
- 编辑角色描述词，加入更具体的外貌特征
- 尝试不同的图片模型（如 Gemini 3 Pro 在角色一致性上表现较好）

### Q2: 分镜画面与剧本描述不符？
- 查看分镜面板的 **AI 数据**，检查图片 Prompt 是否准确
- 手动编辑 Prompt 后重新生成
- 确保角色和场景资产已正确关联到面板

### Q3: 视频质量不理想？
- 尝试更换视频模型（不同模型擅长不同风格）
- 使用 **首末帧模式** 获得更好的画面连贯性
- 调高分辨率（部分模型支持 1080p）

### Q4: 配音语速/语气不自然？
- 调整语速参数（默认 +50%，可改为 +30% 或 +70%）
- 设置情感提示词（如"温柔"、"愤怒"、"兴奋"）
- 调整情感强度（0.1 轻微 ~ 1.0 强烈）

### Q5: 如何获得最佳效果？
1. **文本质量**：输入的小说/剧本文字越优质，AI 分析结果越好
2. **参考图选择**：花时间在角色/场景资产上，这是后续质量的基础
3. **模型搭配**：不同环节可以用不同模型，发挥各自优势
4. **迭代改进**：先跑通全流程，再回头优化不满意的部分
